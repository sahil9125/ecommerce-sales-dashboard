# ecommerce-sales-dashboard
An interactive E-commerce Sales Dashboard built using Power BI and Excel to analyze sales performance, profit, orders, customer insights, and business trends through dynamic visualizations and KPIs.
## Project Overview

This project is an interactive E-commerce Sales Dashboard created using Power BI and Excel to analyze store sales performance and business insights. The dashboard helps track important KPIs such as Total Sales Amount, Profit, Quantity Sold, and Average Order Value (AOV).

## Key Insights & Analysis

* Analyzed overall sales performance and profit trends.
* Identified Top 5 States generating the highest sales.
* Analyzed Top 5 Customers based on purchase amount.
* Compared Profit by Month to identify profitable and loss-making months.
* Examined Profit by Sub-Category to understand best-performing products.
* Analyzed Category-wise sales distribution.
* Compared different Payment Modes such as COD, UPI, Debit Card, Credit Card, and EMI.

## Dashboard Features

* Interactive filters and quarter-wise analysis.
* KPI cards for quick business insights.
* Bar charts, donut charts, and monthly profit analysis visuals.
* Dynamic and user-friendly dashboard design.

## Tools Used

* Power BI
* Microsoft Excel

## Business Questions Solved

* Which state generates the highest sales?
* Which customers contribute the most revenue?
* Which product sub-category earns maximum profit?
* Which months show highest and lowest profit?
* Which payment method is used most by customers?
* Which category has the highest sales contribution?

This project demonstrates skills in data cleaning, data visualization, dashboard creation, business analysis, and reporting using Power BI and Excel.
